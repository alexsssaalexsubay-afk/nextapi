# 小白常见问题（FAQ）

> 与 [`GLOSSARY-ZH.md`](./GLOSSARY-ZH.md)、[`FLOW-ZH.md`](./FLOW-ZH.md) 互补。回答尽量短。

---

## 通用

**Q：我完全不会编程，能用 NextAPI 吗？**  
A：可以。用浏览器打开 **客户控制台** 拿 Key；生成可以交给同事用文档里的工具或 API。你要做的是 **保管好 Key** 和 **认准网址**。

**Q：nextapi.top 和 app.nextapi.top 有什么区别？**  
A：前者偏 **官网展示**；后者是 **登录后的工作台**（密钥、任务等）。

**Q：api.nextapi.top 在浏览器里打开是一串 JSON 或 404，正常吗？**  
A：API 是给 **程序** 用的，不是给人看的网页。健康检查可以访问文档里写的 `/health` 路径。

## 积分与失败

**Q：任务失败还会扣钱吗？**  
A：一般会 **释放预留** / 按规则退款；若账单异常，记下 **任务 ID** 和时间找支持，并看 [`OPERATOR-HANDBOOK.md`](./OPERATOR-HANDBOOK.md)。

**Q：402、429 是什么意思？**  
A：常见 HTTP 状态：**402** 多与余额/花费策略有关；**429** 表示请求太频繁被限速。具体文案以后端返回为准。

## 密钥与安全

**Q：Key 泄露了怎么办？**  
A：立刻在控制台 **吊销** 旧 Key，新建一把，并检查账单与任务记录。

**Q：可以把 sk_ 密钥发给客服吗？**  
A：**不要。** 只能自己或可信同事在 **安全渠道** 使用；客服排查问题用 **任务 ID、时间、报错截图** 即可。

---

## 部署与本地

**Q：我克隆了仓库，怎么在电脑上跑起来？**  
A：看根目录 [`README.md`](../README.md) 的 **Quick start**：需要安装 pnpm、Go、Docker（用于数据库）等，再 `make dev`。跑不起来把 **完整报错** 发给懂行的同事。

**Q：改网页要不要动 backend？**  
A：只改展示或文案时，经常 **只动** `apps/*` 或 `packages/ui`；**改接口或计费** 一般要动 `backend` 并迁移数据库。

---

## 视频模型

**Q：客户选的 `seedance-2.0` 和火山控制台里一长串英文模型名是什么关系？**  
A：网关会 **自动映射** 到 Dreamina 等接入点（有图/无图、标准/快速）。运维核对环境变量见 [`OPERATOR-HANDBOOK.md`](./OPERATOR-HANDBOOK.md)。

## 文档找不到

**Q：对外用户文档和仓库里的 docs 有什么区别？**  
A：**`docs-site/`** 面向产品说明（可发布）；**`docs/`** 含运维、零基础、流程等 **全站运营** 必读。索引见 [`README.md`](./README.md)。
