## 产品与业务

| 中文 | 说明 |
|------|------|
| **NextAPI** | 本项目：面向客户的「视频生成 API 网关」。客户用你的平台发请求生成视频，而不必直接对接火山引擎控制台。 |
| **客户 / 用户** | 在 **Dashboard**（`app.nextapi.top`）注册、充值、创建 API Key 的一方。 |
| **组织（Org）** | 计费与权限的归属单位；一个组织可有多个成员、多把 Key、共享积分余额。 |
| **积分 / Credits** | 早期文档中的叫法，现已废弃。系统底层统一使用 **USD Cents (美分)**，前台直接展示美金（如 $10.00）。10美金在库里存为 1000 美分。 |
| **API Key** | 一串以 `sk_live_…` 开头的密钥，用来调用 `api.nextapi.top`。等同于「通行证」，**绝不能**发到公开网页或聊天群。 |
| **任务 / Job** | 一次视频生成请求在后端对应一条异步任务：排队 → 运行 → 成功或失败。 |

### 三种常见凭证（混用会报错）

| 前缀 / 形式 | 谁用 | 典型场景 |
|-------------|------|----------|
| **`sk_live_…`** | 客户业务侧 | `POST /v1/videos`、管理自己的密钥与 Webhook。 |
| **`ak_…`** | 管理员自动化 | 管理类 API（与客户 `sk_` 路由组不同）。 |
| **`ops_…`（运维会话）** | 登录 Admin 网页后 | 浏览器存会话，调 `/v1/internal/admin/...`；高危操作要 **OTP**。 |

## 网站与入口

| 中文 | 网址（示例） | 说明 |
|------|----------------|------|
| **营销站** | `nextapi.top` | 介绍产品、价格、文档入口。 |
| **客户控制台（Dashboard）** | `app.nextapi.top` | 登录后管理 Key、看任务、试用生成。 |
| **管理后台（Admin）** | `admin.nextapi.top` | 给**运营/老板**：调账、暂停组织、看审计等；不是普通客户用的。 |
| **API** | `api.nextapi.top` | 程序调用的接口地址（HTTPS）。 |

## 技术名词（白话版）

| 词 | 白话 |
|----|------|
| **API** | 程序之间对话的「窗口」。你或你的程序员向固定网址发结构化请求，服务器返回结果。 |
| **HTTPS** | 加密传输，地址以 `https://` 开头。 |
| **异步** | 提交后不必一直等着；先发请求拿「任务号」，再轮询或等 Webhook 通知完成。 |
| **Webhook** | 任务状态变化时，**你的服务器**提供一个网址，NextAPI **主动 POST 通知**你（需验签）。 |
| **轮询（Poll）** | 每隔几秒问一次「好了吗」，直到任务成功或失败。 |
| **环境变量（.env）** | 服务器上的配置清单（密码、密钥、开关），**不**写进代码里，避免泄露。 |
| **数据库迁移（Migration）** | 一组按顺序执行的 SQL 文件，用来**升级表结构**；新功能常依赖新迁移。 |
| **VPS** | 一台租用的云服务器；本项目里 API 跑在香港机房的 VPS 上。 |
| **DNS** | 把域名指到哪个服务器；例如 `api.nextapi.top` 指向你的 VPS IP。 |
| **Clerk** | 第三方登录服务；Dashboard/Admin 用它做注册、登录。 |
| **Cloudflare Workers / Pages** | 边缘上运行的网站托管；营销站、Dashboard、Admin 前端部署在这类服务上。 |
| **R2** | Cloudflare 的对象存储，类似「网盘 bucket」，可存生成的视频文件（按你实际配置为准）。 |
| **OpenAPI** | 机器可读的 API 说明书；本仓库在 `backend/api/openapi.yaml`。 |
| **Seedance / 火山引擎 / Ark** | 字节跳动火山引擎上的视频大模型能力；NextAPI 在 **live** 模式下把请求**转发**到对应模型接入点。 |
| **Dreamina** | 火山侧产品/规格命名；2.0 档位在代码里映射为四条 `Dreamina-Seedance-2.0-*` 模型名（见运营手册）。 |

## 任务与错误

| 词 | 说明 |
|----|------|
| **401 / 403** | 未登录或无权访问；检查 Key 是否过期、是否带对请求头。 |
| **402** | 余额不足等业务拒绝；需充值或降低并发。 |
| **429** | 请求太频繁；触发了限速。 |
| **model not found** | 上游不识别模型 ID；检查运营手册里的 `SEEDANCE_MODEL` / `SEEDANCE_MODEL_MAP` / Dreamina 四档配置。 |
